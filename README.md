# Klassifikation von Zufallsdaten mit Decision Tree & KNN

Dieses Projekt erzeugt einen künstlichen Datensatz mit drei Klassen und zwei Merkmalen pro Klasse und vergleicht zwei Klassifikationsverfahren:

- **Decision Tree Classifier**
- **K‑Nearest Neighbors (KNN) Classifier**

Die Daten werden zufällig erzeugt, wobei jede Klasse eine multivariate Normalverteilung mit unterschiedlichen Mittelwerten und Kovarianzen besitzt, um eine Überlappung der Klassen zu erzeugen.
